# Local Tool Showcase

这是一个静态网页练手项目，用来介绍 LocalSend 和 Stirling PDF。

## 本地预览

直接打开 `index.html` 即可。

## GitHub Pages 发布

1. 在 GitHub 新建仓库，例如 `tool-showcase`。
2. 把本文件夹里的内容上传到仓库根目录。
3. 打开仓库 `Settings` -> `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 GitHub 生成访问地址。
